定義物件其實跟我們定義變數的時候差異不大

# 程式碼

```javascript
var family = { // 這樣就可以直接定義物件，稱為物件實字
    name: '小明家', // property: value,
    deposit: 1000, // property2: value2,
    members: {
        mom: '老媽',
        ming: '小明',
    }
callName: function() {
    console.log('連絡小明家')
  }
};

console.log(family);
```
