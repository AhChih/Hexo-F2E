CSS 權重 (css specificity)

發表於 2020-05-28 更新於 2020-07-25 分類於 CSS

所謂 CSS 的權重就是看誰擁有 優先權

說明：

  • 二個 CSS 擁有相同的權重，後來寫的可以蓋掉前面的
  • 兩個選擇器同時作用在同一個元素，看誰的權重比較高，高的可以蓋掉前面的

權重比較

那麼我們就來比較一下各權重的分數吧

分數越高的代表越重，可以蓋掉前面的

全站預設 (0分)

元素 (1分)

例如 div, p, ul, li, span 這些

class (10分)

例如 div class="header"

id (100分)

例如 div class="id"

inline style (1000分)

例如 div style="color:red;"

虛擬類別 (10分)

例如 :hove, :active,

important (終極奧義無限大)

只有後面寫的 important 才能蓋掉
